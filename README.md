# API Documentation

## Course End point

### Get Course All

 - [GET] `localhost:9000/api/courses`

```
    {
        course_title: "The Complete JavaScript",

```

### Get Courses by ID

 - [GET] `localhost:9000/api/courses/:id`

### Post Course
